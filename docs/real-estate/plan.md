# 🏠 부동산 인텔리전스 — 전체 기획 (2026-07-11)

주식·코인에 이어 포트폴리오 3번째 축. **주식에서 검증한 프레임(사이클·밸류·수급·심리 4축 + 정직 캐비엇)을 부동산 언어로 번역**한다.

## 4축 번역표

| 축 | 주식에서 | 부동산에서 |
|---|---|---|
| 사이클 | 4계절·막스 시계추·BTC 사이클 | **벌집순환모형**(가격×거래량 6국면) + 공급 파이프라인(인허가→착공→입주, 2~3년 시차) |
| 밸류에이션 | PER·PBR·PEG | **전세가율**(내재 사용가치)·**PIR**(소득 대비)·**월세수익률 vs 금리**(위험프리미엄) |
| 수급 | 외인/기관 순매수 | 입주물량 vs 수요·**미분양(부동산판 재고 적체)**·거래량·경매 낙찰가율 |
| 심리 | 공포탐욕·칵테일파티 | KB 매수우위지수·전망지수 |

## 단계 로드맵

- **Phase 0** ✅ 데이터 소스 전수 실측 (2026-07-11 완료 — 판정표는 context-notes.md)
- **Phase 1** 🏠 시장 대시보드("부동산판 Fed Watch"): 매매/전세 지수 + 기준금리·주담대금리 오버레이 + 미분양 + 실거래가격지수. **ECOS 단독으로 구축 가능**(실키 필요·즉시발급)
- **Phase 2** 🐝 벌집순환모형 내비게이터: 가격×거래량 6국면 자동 판정(로테이션 시계 패턴) + 공급 파이프라인(주글라르 패턴). R-ONE 키 필요(지역 세분)
- **Phase 3** 🔍 아파트 리서치: 단지 검색→실거래 추이(매매·전세 나란히)+밸류 3축(전세가율·PIR·수익률-금리 스프레드)+지역 피어 비교. **data.go.kr RTMS 키 필요**
- **Phase 4** 💼 포트폴리오 통합: 내 부동산 등록(단지+평형)→실거래 시세 자동 추적→총자산(주식+코인+부동산) 비중. 유동성·거래비용 차이 정직 고지, 리밸런싱엔 비중 참고만
- **Phase 5** 🎓 교육: 갭투자 레버리지 시뮬(역전세 위험=음의 복리 패턴)·LTV/DSR 계산기·"소액 부동산=리츠" 다리(GICS 부동산 섹터·리츠 배당 기보유)·자산군 장기수익률 실측 비교

## ⛔ 하지 않는 것(원칙)

- 지역 매수 추천(투자자문 리스크·개인 상황 의존) — **위치·밸류 관측 도구까지만**
- 세금 계산기(규칙 수시 변경→하드코딩 필패) — 개념 교육+공식 계산기 링크
- 호가 데이터(실거래 아님·스크랩 ToS) — **실거래가만**
- AI 가격 예측 — 결정론 지표만
