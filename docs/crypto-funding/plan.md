# 🪙 크립토 펀딩비·OI 과열 레이더 — 킬러 로드맵 #6

## 한 줄
BTC·ETH·SOL 무기한 선물의 **펀딩비(funding rate)**와 **미결제약정(OI)**으로 레버리지 과열/청산 위험을 재는 레이더. 코인 랩 ₿ 뷰(ETF 플로우 다음).

## 왜 (확정적으로 잃지 않게)
- 현물 가격만 보면 레버리지 쏠림(청산 캐스케이드 연료)이 안 보인다. 펀딩비 = 롱/숏 어느 쪽이 과밀한가의 실시간 온도.
- 펀딩비 高양(+) = 롱이 숏에 이자 지급 = **롱 과열**(작은 하락에도 연쇄청산 위험) / 펀딩비 음(−) = 숏 과밀 = **역발상 반등(숏스퀴즈) 연료**.
- OI 급증 + 펀딩 高 = 레버리지 빌드업(사이클 후반 froth). 경보만·매수/매도 지시 아님.

## 데이터 (Binance 무료·무키·Phase 0 실측 완료)
- `fapi/v1/premiumIndex?symbol=` → markPrice·lastFundingRate(현재 8h 펀딩).
- `fapi/v1/fundingRate?symbol=&limit=1000` → 펀딩 이력(8h 간격 ~333일) → 백분위·30일 평균.
- `fapi/v1/openInterest?symbol=` → 현재 OI(코인 수).
- `futures/data/openInterestHist?symbol=&period=1d&limit=30` → OI 명목가($) 30일(Binance 1d는 30일 상한 실측).
- ⚠️ Binance가 일부 데이터센터 IP 차단 가능 → 라이브 검증 필수, 막히면 Bybit 폴백.

## 계산 (결정론)
- 연율화 펀딩 = lastFundingRate × 3 × 365(8h당 3회/일).
- 백분위 = 현재 연율화의 ~1년 이력 내 위치.
- OI 30일 변화% = (최신 sumOpenInterestValue − 30일전)/30일전.
- 판정: `long_hot`(연율 ≥30%·심함 ≥60%) / `short_skew`(연율 ≤−5%) / `neutral`. + OI 30일 +25%↑=레버리지 급증 보강.

## UI
- 코인별 카드: 연율 펀딩(게이지)·백분위·30일 평균·OI 명목가·OI 30일 변화·펀딩 스파크라인·판정 배지.
- 종합 배너(가장 과열 코인 기준) + 교육: "펀딩 高=롱 과열=청산 위험 / 음(−)=숏 과밀=역발상".
- 캐비엇: 코인 ≤5% 가드·펀딩은 후행/단기 지표·매수매도 지시 아님·과열이 곧 하락 아님(고펀딩 지속 랠리도 흔함).
