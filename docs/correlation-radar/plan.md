# 🕸️ 이종 자산 상관 수렴 레이더 — 킬러 로드맵 #5

## 한 줄
주식(SPY)·채권(TLT)·금(GLD)·달러(UUP)·BTC 5축의 **일별 수익률 상관**을 롤링으로 재고, **"평시 대비 최근 30일 상관 급등"**(위기 시 모든 자산이 함께 떨어지는 ρ→1 수렴)을 경보로. 분산투자의 착시를 깨는 방어 레이어.

## 왜 (더 벌게 X → 확정적으로 잃지 않게 O)
- 학생은 "주식+채권+금+코인 = 분산됐다"고 믿지만, 위기(2008·2020·2022)엔 상관이 1.0으로 수렴해 **동반 폭락** → 분산 효과 소멸.
- 앱의 상관관계 매트릭스(corr-matrix)는 **내 개별 주식 간** 상관(전부 기술주면 가짜 분산). 이건 **이종 자산군(asset class) 간** 상관 = 포트폴리오 전체의 진짜 방어력.
- 경보만 — 제미나이의 "기계적 현금화"는 거부. 관측·교육.

## 데이터 (신규 수집 최소)
- 5축 캔들: `techChartData.getTechCandles(ticker,'US','D')` SSOT 재사용 — SPY·TLT·GLD·UUP·BTC-USD (Phase 0 실측: 501·501·501·501·731봉 확보 확인).
- HY 스프레드(위기 맥락 보조): `/api/macro-weather` 자기호출(공개·12h 캐시·graceful) — 읽기만.

## 계산 (결정론·순수)
1. 각 자산 일별 종가 → date→close 맵 → **공통 거래일** 정렬(BTC 주말봉은 자동 제외).
2. 일별 수익률 = (close[t]−close[t-1])/close[t-1].
3. 최근 30 거래일 = recent, 그 이전 ~200 거래일 = baseline(최소 60).
4. 10개 쌍 각각 피어슨 r (recent·baseline·delta).
5. **평균 쌍상관**(signed mean of 10 pairwise r) recent vs baseline → 분산 효과 척도(높을수록 분산 약화).
6. 경보:
   - `converging` — meanRecent ≥ 0.40 AND delta ≥ +0.20 (전반 상관 급등 = 수렴)
   - `watch` — delta ≥ +0.15 OR meanRecent ≥ 0.35
   - `calm` — 그 외
   - HY 스프레드 높으면(≥4.0%) 위기 맥락 보강 코멘트.

## UI (막스 시계추 탭, MarksCycle 하단)
- 헤더 + 경보 배지 + 평시↔최근 평균 상관 게이지 대비.
- 5×5 상관 히트맵 2개(평시/최근) 또는 쌍별 델타 바.
- 달러(UUP)는 위기 시 오히려 디커플(안전자산 강세)되는 진짜 헤지 — 교육 포인트.
- 캐비엇: 상관은 후행·과거 30일 창·매수/매도 지시 아님·현금화 강요 아님.
