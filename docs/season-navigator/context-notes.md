# 🧭 4계절 매크로 내비게이터 — 컨텍스트 노트

작업 중 내린 결정과 그 근거를 계속 적립한다.

## 결정 1. 새 판정기 대신 macro-regime SSOT의 뷰로 구현
- 이유. 앱에 이미 매크로 판정이 둘 있음(`/api/macro-regime` 6 phase, `/api/macro-weather` 3 날씨). 2026-06-07에 화면 간 매크로 모순을 SSOT로 막 통일함(제2원칙).
- Gemini 스펙대로 세 번째 독립 판정기를 더하면 "맑음 + 금리 고점 + 가을 스태그"가 동시에 떠 모순 부활.
- 따라서 계절은 macro-regime을 읽는 또 하나의 소비자(뷰)일 뿐. 매크로 결론은 영원히 하나.

## 결정 2. 진실 모델 = 2×2 정태 사분면 + 계절 별칭 (사용자 확정)
- 업로드 다이어그램은 성장×물가 2×2 정태(골디락스/인플레/스태그/리세션).
- Gemini가 덧씌운 봄여름가을겨울은 동태 사이클이라 방향(모멘텀) 추론이 필요해 fragile.
- 2축 부호로 칸이 결정론적으로 정해지는 2×2를 엔진으로, 계절명은 친근한 별칭(스킨)으로.

## 결정 3. 현금은 조언 텍스트로만 (사용자 확정)
- investments 테이블에 현금 개념이 없음(ticker·purchase_price·quantity·lynch_category뿐). 모든 비중은 보유 주식만으로 100%.
- Gemini의 Seasonal Alignment Score 공식 핵심항 `현금비중 × 현금점수`는 데이터가 없어 정의 불가.
- 점수는 보유 주식의 계절 적합도만. 권장 현금 비중은 "직접 확인" 안내 문구로만. 스키마 변경 0, 거짓 데이터 0.

## 결정 4 (Phase 0 결과). 성장축 = OECD CLI
- Gemini가 제안한 한국 재고순환선(출하-재고)은 FRED에서 stale(KORPROINDMISMEI 2024-03, BSCICP03KRM665S 2023-12) → 사실상 환각.
- 대체재로 OECD 경기선행지수 CLI 확인. 한국 KORLOLITOAASTSAM·미국 USALOLITOAASTSAM 둘 다 신선(2026-04).
- CLI는 경기 전환점 포착용으로 설계 → 성장축에 이상적. 레벨(100 기준)=추세 상회/하회, 기울기=방향.
- 실측(2026-04). 한국 102.50 상승추세, 미국 100.85 상승추세 → 둘 다 성장 확장 국면.
- 물가축은 macro-regime의 CPI YoY 재사용.

## 사분면 매핑 초안 (Phase 1에서 확정)
- 성장↑ × 물가↓ = 골디락스(봄/여름 경계) — 주식 최적
- 성장↑ × 물가↑ = 인플레이션(여름) — 시클리컬/에너지
- 성장↓ × 물가↑ = 스태그플레이션(가을) — 방어+현금, 주식 worst
- 성장↓ × 물가↓ = 리세션(겨울) — 방어주(통신/유틸/필수소비재)
- 지표 상충/경계 = 간절기(SHOULDER) → 보수적 운용

## 결정 5 (확장). 시장별 계절 분리 — 성장축만 시장별, 물가축은 글로벌 공통
- 사용자 요청. 국장/미장 계절 분리.
- Phase 0와 동일 검증. 한국 CPI(KORCPIALLMINMEI·CPALTT01KRM659N) FRED에서 stale(2023-11) → 완전 분리된 한국 계절은 물가축이 막힘.
- 한국 CLI(KORLOLITOAASTSAM)·정책금리(IRSTCI01KRM156N)는 신선(2026-04) → 성장축은 시장별 가능.
- 절충. 한국 계절 = 한국 CLI(성장) × 글로벌 물가(macro-regime). 한국은 소규모 개방경제로 글로벌 물가를 수입하고 한은이 연준 추종 → 물가축 글로벌 사용은 방어 가능. 단 "물가=글로벌 기준" 명시.
- 구현. fetchCli(US·KR) 병렬, seasonOf로 각각 판정. 종목은 시장(6자리/market)으로 갈라 자기 시장 계절로 채점(holdingFit 추출해 SSOT 공유). UI는 메인 다이어그램=미국 앵커 + 시장별 배지 strip + 종목행 국기.
- 메인 다이어그램·행동가이드는 미국(글로벌 앵커) 기준 유지(물가축이 글로벌이라 미국이 앵커로 일관).
- 검증. 현재 미 CLI 100.85↑·한 CLI 102.50↑·물가 과열 → 양쪽 여름(동조). 갈리면 strip에 ⚠️ + 종목별 분리 채점.
