# 🚨 DART 희석 경보 — 설계

> 7대 킬러 로드맵 #4 (2026-07-18). 유상증자·CB·BW·EB·감자 = 개미 계좌를 실제로 깨뜨리는 이벤트 1순위인데 앱이 침묵하던 영역. 보유 KR 종목의 DART 공시를 감시해 배너로.

## Phase 0 실측 (보유 KR 전 종목 180일 스캔 — 키워드 원시 매칭 20건)
| 발견 | 처리 |
|---|---|
| SK하이닉스·KB금융 "유상증자결정(**종속회사/자회사**의 주요경영사항)" | **제외** — 자회사 증자는 모회사 주주 희석 아님(우량주 오탐의 주범) |
| SK하이닉스 "자기교환사채 **만기전취득**결정" | **제외** — 오버행 축소(호재)인데 '교환사채' 키워드에 걸림. 취득·상환 제외 |
| 대한광통신 "[기재정정]주요사항보고서(유상증자결정)" 02-23 | ✅ 진짜 희석 경보(본체 유상증자) |
| LG전자 "주요사항보고서(감자결정)" 02-12 | ✅ 경보 — 단 유상감자(환원)/무상감자(결손) 구분은 제목 불가 → 원문 확인 안내 |
| 발행가액확정·청약결과 등 후속 공시 | 노이즈 — **주요사항보고서(…결정) 본공시만** 채택 |
| ETF·우선주는 corp_code 없음 | graceful 스킵(STOCK 필터 + null 허용) |

## 판정 규칙 (결정론)
```
CORE  = /주요사항보고서\((유상증자결정|전환사채권발행결정|신주인수권부사채권발행결정|교환사채권발행결정|감자결정)\)/
제외  = /종속회사|자회사|타법인/  또는  /취득|상환/
유형  = 유상증자 | CB | BW | EB | 감자
```
- 창: 최근 180일(진행 중 이벤트 커버), UI는 30일 이내 🚨 강조.
- 공시 원문 링크: `https://dart.fss.or.kr/dsaf001/main.do?rcpNo={rcept_no}`

## 구현
- `/api/dilution-alert`(auth): 보유 KR STOCK(6자리) → 종목별 공유 캐시 `dilution-v1:{ticker}:{일자}` 24h(제2원칙 — 학생 간 공유) → getCorpCode + `list.json` 180d → 필터 → 유형 분류.
- `DilutionAlertBanner.tsx`: 대시보드 상단(타점 워처 아래) — 경보 없으면 렌더 0. 칩: 유형 배지+종목+날짜+DART 원문 링크.
- 교육: 유상증자=주식수 증가로 기존 주주 지분 희석(주주배정/3자배정 성격은 원문 확인) · CB/BW/EB=잠재 물량 오버행 · 감자=유상/무상 구분 필수 · ⛔ 매도 지시 아님(성장 투자용 증자도 있음 — 용도·규모를 원문으로).
