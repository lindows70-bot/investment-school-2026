# 📜 채권 축 신설 — 킬러 로드맵 #7 (마지막)

## 한 줄
채권을 앱의 4번째 자산군으로 신설. **듀레이션 나침반**(금리 방향×수익률곡선×크레딧 스프레드 → 단기/중기/장기채·국채/크레딧 가이드) + **채권 ETF 현황** + **금리 ±1%p 듀레이션별 손익 시뮬**. `/bonds` 독립 페이지 = 채권 자산군 진입 화면.

## 왜
- 주식·부동산·코인은 있는데 채권 축이 0 → 자산군 밸런스 미완성. 금리 국면에 맞는 채권 포지션(장기채로 금리인하 베팅 vs 단기채 방어)은 방어적 자산배분의 핵심.
- "금리는 채권 가격의 중력" — 듀레이션이 길수록 금리 변동에 민감. 학생이 이 관계를 손익 숫자로 체감.

## 데이터 (신규 수집 최소)
- 금리 국면: `/api/macro-regime` SSOT 재사용(rateDir·yieldCurve·hySpread·fedRate — 제2원칙, 자기호출·읽기만).
- 채권 ETF 가격: `techChartData.getTechCandles` SSOT(US=SHY·IEF·TLT·LQD·HYG, KR=148070 KOSEF 국고채10년) → 1M/3M/1Y 수익률.
- 듀레이션(modified duration): ETF의 잘 알려진 고정 특성값(참조 상수 — FRED 폴백처럼 허용, UI에 '대략' 명시).

## 계산 (결정론)
- **듀레이션 나침반**: rateDir=cut→장기채 유리(금리↓=긴 듀레이션 최대 이득) / hike→단기채 방어 / hold→중기채 균형. + 수익률곡선 역전(<0)=침체 선행 경고 + HY 스프레드 ≥4%=크레딧 회피·국채 선호 / <4%=크레딧 캐리 가능.
- **금리 ±1%p 손익 시뮬**: ΔP ≈ −ModDur × Δy(수정듀레이션 근사). 금리 −1%p→+ModDur%, +1%p→−ModDur%. TLT(dur~17)=±17%, SHY(~1.9)=±1.9% → "긴 채권일수록 금리에 크게 흔들린다" 체감.
