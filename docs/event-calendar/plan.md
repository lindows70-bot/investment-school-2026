# 📅 내 종목 이벤트 캘린더 — 설계

> 7대 킬러 로드맵 #3 (2026-07-18). 보유 종목의 어닝 D-day·배당락·배당 지급을 한 캘린더에 — 갭 리스크(매매 플랜 카드의 경고)를 '날짜'와 연결하는 실전 도구 + 배당 현금흐름 가시화.

## 데이터 (전부 기존 검증 소스)
| 항목 | 소스 | 비고 |
|---|---|---|
| 어닝 일자 | Yahoo `quoteSummary calendarEvents.earnings.earningsDate[0]` | sectorEngine 검증 패턴. **KR 미제공(정직 캐비엇)** |
| 배당락·지급일 | `calendarEvents.exDividendDate / dividendDate` | KR은 .KS→.KQ 폴백, 대부분 연 1회 |
| 연 배당금 | `summaryDetail.dividendRate`(주당·연) | 배당 없는 종목 null |
| 월별 현금흐름 추정 | Yahoo `chart(events:'div')` 최근 12개월 지급 이력 → **12개월 앞으로 투영** | "과거 패턴 기반 추정" 명시 |

## 산출 (결정론)
- **이벤트 타임라인**: 향후 90일 — 🎯실적(D-day)·💰배당락·💵지급일, D-day 오름차순.
- **연간 예상 배당** = Σ 보유수량 × dividendRate × 환율 + 종목별 표(수량·주당·연간₩·시가배당률).
- **월별 예상 배당 바차트**: 과거 12M 지급 이력의 (월, 주당액)을 다음 해 같은 달로 투영 × 수량 × 환율.
- 대상: 보유 US·KR 주식+ETF (크립토 제외 — 어닝·배당 없음). ETF는 어닝 스킵·배당만.

## 정직 캐비엇
1. 어닝 일자는 기업이 수시로 변경(±수일) — 확정 공시가 아님.
2. KR 실적 발표일은 무료 소스 미제공(DART 잠정실적 공시는 사후) — 어닝 타임라인은 US 위주.
3. 월별 배당은 과거 12개월 지급 패턴의 투영(증배·감배·일정 변경 미반영), 환율은 현재 기준.

- 브리핑 `①½ 이번 주 이벤트`: 같은 API에서 D-7 이내 이벤트만 칩으로(같은 SSOT — 제2원칙).
