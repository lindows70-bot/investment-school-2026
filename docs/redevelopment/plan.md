## 데이터 구조
- 6,581 고시 → **4,066 고유 구역**(PRJC_CD). 각 구역 다수 고시(지정·변경·폐지).
- 필드: LOGVM(대부분 '서울특별시'), SCLSF(사업유형), RGN_NM(구역명), PSTN_NM(위치 지번), AREA_EXS(면적), RPT_TYPE(신설/변경/폐지/실효/정정/기정), RPT_MNG_CD(내 8자리 고시일).
- **자치구**: LOGVM은 88% '서울특별시'라 무용 → PSTN_NM 지번에서 구 파싱 + **동→구 맵을 데이터에서 학습** → 89% 커버(3630/4066). 서울 25구 화이트리스트로 오염(조로구·불광지구) 필터.
- **최신 상태**: PRJC_CD 그룹핑 → 고시일 최신 고시의 RPT_TYPE. 폐지/실효 제외 = 활성.
- **유형 그룹**: 재개발/재건축/도시환경정비/주거환경/기타(SCLSF 키워드).
