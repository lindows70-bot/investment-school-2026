# 🚨 개인 빚투 레이더 & 반대매매 시뮬레이터 — 기획 (2026-07-03)

> 사용자 요청: 개인투자자 빚투 지표로 위험성 교육 + 실투자 참고. 제미나이 기획안 참고하되 더 멋지게.

## 제미나이 안 비판적 수용 (교정 3건)

| 제미나이 안 | 문제 | 우리 설계 |
|---|---|---|
| 4대 지표를 **슬라이더로 입력** | 신용잔고는 매일 공표되는 **실데이터** — 학생이 지어내게 하면 Zero-Input 위반 + 교육 왜곡 | **실데이터 자동 경보판** (슬라이더 제거) |
| "실시간 슬라이더 조절하며 위험 판별" | 위험 판정도 실측 기준으로 결정론 산출해야 | 신용/예탁금 비율 + 신용잔고 역사적 백분위로 3단계 신호등 자동 판정 |
| 반대매매 시뮬레이터 | ✅ 좋은 기획 — 이건 가정 실험이라 인터랙티브가 정당(레버리지 시뮬·디페깅 시뮬과 동형) | 채택 + 담보유지비율 140%·T+2 반대매매 수학 정확 구현 |

## 구성 (3단)

### ① 빚투 경보판 (실데이터 자동 — Zero-Input)
- **신용잔고 추이 차트**(일별, 수년) + 고객예탁금 오버레이(이중축)
- **빚투 비율 게이지**: 신용잔고÷예탁금. 판정 = 역사적 분포 기반(최근 3년 백분위): ≥80퍼센타일 🔴위험 / ≥60 🟡주의 / 그외 🟢안정 — 절대 임계 하드코딩 대신 데이터 분포로(PSR 상대판정과 같은 철학)
- 신용잔고 증감 속도(20일 변화율): 급증 = 과열 신호 배지
- **역발상 교육**: 신용잔고 급감(반대매매 청산) = 역사적으로 바닥 신호이기도 — 2020.3, 2024.8 실데이터로 마킹

### ② 반대매매 수학 시뮬레이터 (가정 실험 — 인터랙티브 정당)
- 입력: 매수금액·신용융자 비율(40~60%)·담보유지비율(140% 표준)
- 출력: **"몇 % 하락하면 마진콜 → T+2 아침 하한가 반대매매"** 임계 주가 + 손실 증폭 배수
- 수식: 담보비율 = (주식평가액+현금)÷융자금. 임계하락률 = 1 − 1.4×융자비율... 정확 도출해 구현
- 레버리지 시뮬레이터(음의 복리)와 교차 링크

### ③ 생존 가이드 (결정론 텍스트)
- 현재 경보 레벨별 행동 지침 + "빚투 잔고가 사상 최고 = 시장이 가장 낙관적인 순간 = 달리오 버블 지표의 '신용 팽창'과 동일 신호" — 레이 달리오 탭·버블 지표와 SSOT 연결

## API 설계
- `/api/leverage-radar` (공개 · 12h 캐시 `leverage-radar-v1`): 네이버 예탁금 페이지 N페이지 파싱(EUC-KR cp949 디코딩 — investorDealTrendDay와 동일 패턴) → { series: [{date, deposit, margin, ratio}], current: {…, percentile, level}, usMargin?: … }
- 판정 결정론(백분위) — AI 미사용

## 검증 계획
